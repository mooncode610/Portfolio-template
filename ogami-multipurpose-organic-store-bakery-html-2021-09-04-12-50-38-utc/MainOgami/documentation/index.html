Template Installation

FTP Upload

  • Open up your FTP manager and connect to your hosting
  • Browse to required directory (Normally public_html)
  • Upload the files inside Ogami folder.

 

Pages

  • index-2.html is Home Page Version 2
  • index-3.html is Home Page Version 3
  • index-4.html is Home Page Version 4

  • shop_checkout.html is Shopping Checkout Page

  • shop_checkout.html is Shop Page With Filter Option

  • shop_detail_fullwidth.html is Product Details Fullwidth Page

  • shop_grid+list_fullwidth.html is Shop Page Fullwidth With Sidebar
  • shop_wishlist.html is Shop Page With Wishlist Products
  • blog_detail_sidebar.html is Blog Detail Version With Sidebar

  • blog_grid_2col.html is Blog Grid Version With 2 Column
